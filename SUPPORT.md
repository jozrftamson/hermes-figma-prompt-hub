# Support

Use GitHub issues and discussions for support.

## Questions and Ideas

Use GitHub Discussions for:

- ideas
- help wanted topics
- show and tell
- general questions

## Bugs and Feature Requests

Use GitHub Issues for:

- reproducible bugs
- feature requests
- good first issues
- collaborator interest

## Before Asking

Run:

```bash
python scripts/validate_repo.py
```

Include:

- operating system
- Python version
- command that failed
- relevant prompt file or fixture
- expected behavior
