# Human collaborator priorities

The most important collaborators for the next phase are people who can move the project from scaffold to usable integration.

## Priority roles

1. Figma API implementer
   - Builds the importer from `PROMPT/<id>` frames to prompt JSON.
   - Works with fixtures in `evals/cases`.

2. MCP developer
   - Implements `mcp/server.py`.
   - Adds tools for listing, validating, exporting, and eventually syncing prompts.

3. Prompt schema maintainer
   - Evolves `prompts/schema/prompt.schema.json`.
   - Keeps examples, README, and fixtures synchronized.

4. Documentation maintainer
   - Improves setup docs, examples, and workflow guides.
   - Helps new contributors find small tasks.

5. Community maintainer
   - Reviews incoming issues.
   - Runs monthly collaboration updates.
   - Helps keep `good first issue` and `help wanted` labels useful.

## How to invite them

Use `docs/outreach/templates.md` and link directly to:

- `COLLABORATION.md`
- `CONTRIBUTING.md`
- `ROADMAP.md`
- `docs/prompt-catalog.md`
