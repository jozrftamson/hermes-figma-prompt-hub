# Sponsor this project

Hermes Figma Prompt Hub is maintained as an external integration for Figma-based prompt design, validation, and future Hermes MCP workflows.

Sponsorship helps fund:

- Figma API importer work.
- MCP server implementation.
- Prompt schema and validation improvements.
- Documentation and examples for Hermes users.
- Maintenance, issue triage, and contributor support.

## What sponsors support

The current development focus is:

1. A stable Figma frame-to-prompt JSON importer.
2. A usable MCP server for listing, validating, and exporting prompts.
3. Better examples and eval fixtures for prompt quality checks.
4. Documentation that helps Hermes users install the integration without modifying Hermes core.

## For organizations

If your team depends on Figma-to-agent prompt workflows, sponsorship can help prioritize roadmap items that are broadly useful to the open-source project.

Please open an issue for technical discussion before sponsoring work that requires a specific implementation direction.
